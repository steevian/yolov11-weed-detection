# YOLOv11 前端改版交付说明（2026-03-04）

## 1. 改版目标
在不改变检测功能与后端接口契约的前提下，完成前端视觉与结构升级：
- 卡片化主界面
- 左侧导航工作台风格
- 登录页星空动效风格
- 检测页与历史页统一排版
- 用户中心风格统一

## 2. 已完成范围
### 2.1 路由与鉴权
- 默认首页切换到 `dashboard`
- 菜单入口更新为业务语义（图像检测、视频检测、检测记录、用户管理、个人中心）
- 收口鉴权策略：仅 `login/register` 白名单，业务页需登录

### 2.2 主页面（Dashboard）
- 新增工作台：统计卡片、快捷入口、最近检测记录
- 保留现有接口获取统计数据

### 2.3 登录页
- 已迁移为星空卡片动效风格（含角色互动）
- 保留现有登录逻辑：`/flask/login`、token/session 写入、跳转逻辑

### 2.4 检测页
- 图像检测页：重构为「操作卡 + 预览卡 + 结果卡」
- 视频检测页：重构为「操作卡 + 进度条 + 预览卡 + 状态提示」
- 均未改动核心检测逻辑、接口调用、结果渲染流程

### 2.5 历史页
- 图片/视频/摄像记录页统一为卡片化管理页结构
- 保留查询、分页、详情、删除等原有逻辑

### 2.6 用户中心
- 个人中心页面统一卡片化样式
- 用户管理列表页统一管理台样式
- 用户管理弹窗样式升级（结构与交互不变）

## 3. 全局样式收口
- 已在 `src/styles/app.scss` 增加工作台统一样式助手（title/action-card/table-card/pagination）
- 统一卡片圆角、阴影、背景层次、按钮圆角基线

## 4. 验收清单
请按以下顺序验收：
1. 登录页：动效正常、登录成功后进入工作台
2. 侧边导航：所有菜单可进入对应页，权限可见性正常
3. 图像检测：上传、检测、绘框、高亮、结果表格正常
4. 视频检测：上传、进度、结果视频播放、重试提示正常
5. 历史页：筛选、分页、删除、详情/预览正常
6. 个人中心：头像上传、资料修改、回显正常
7. 用户管理：查询、新增、修改、删除、弹窗交互正常
